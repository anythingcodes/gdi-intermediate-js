Our Class Project

Check it out »

This is our class project, but you can make anything you want out of it. Providing you with tools and resources to make what you want of it. CSS and HTML has been provided so we can focus on JS. Today and next week we will be learning how to interact with page elements to add common elements such as filtering and modals. We'll also debug and deal with error-handling to make your JS lives easier. At the end, we'll likely deploy your site to the web so you can start using it and sharing it!
